## Overview

**Import Export Hub** is a modern web platform that simplifies global trade for users. It allows browsing international products, managing exports, and importing items directly into a personal **“My Imports”** section with a single click. The platform is responsive, secure, and ideal for businesses and individuals looking to streamline import-export operations.

---

## Project Description

Import Export Hub enables users to efficiently manage exports, explore global products, and import items into their personal dashboard. Key features include:

- Real-time data synchronization
- Secure user authentication
- Clean, responsive UI
- Private routes for authenticated users
- Seamless importer and exporter workflow

This platform is designed to give both exporters and importers a smooth and efficient experience while managing global trade.

### Install dependencies

npm install

### Start development server

npm run dev

## Features

- User Login/Register with Firebase Authentication
- Google Login support
- Add/Delete/Edit tour packages
- Book tours and view user-specific bookings
- Responsive design (mobile & desktop)
- Dark / Light theme toggle
- Protected routes for private pages

## Tech Stack

- **Frontend:** React.js, Tailwind CSS, DaisyUI
- **Routing:** React Router DOM
- **Backend:** Node.js & Express.js
- **Database:** MongoDB Atlas
- **Authentication:** Firebase Authentication
- **Notifications:** SweetAlert2
- **Animations:** Lottie React
